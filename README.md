# 3D castle Game
## Description 
This game is a first-person game  where the player can freely explore and observe a beautifully designed castle.
## Controllers
- A --> move left
- D --> move right 
- W --> move forward
- S --> move backwards

- right --> move right 
- left --> move left
- up --> move forward
- down --> move backwards

- Space --> jump

- F --> Free mode
